WMS Notes is een hiërarchische (boomgebaseerde) notitie-app.

<b>Opmerking: Omdat deze app nog volop in ontwikkeling is kan de app zich vreemd gedragen of onverwacht crashen.</b>

Features:

• Een hiërarchie met notities en mappen
• Synchronizatie tussen je desktop en Android-telefoon waardoor je offline je notities kunt bewerken
• Markdown voor de opmaak
• Afbeeldingen toevoegen

De applicatie bestaat uit drie delen:

• Een desktop-applicatie voor Linux, Windows en macOS
• Een Android-app
• Een server-applicatie voor Linux, Windows and macOS

WMS Notes is open source! Kijk op https://github.com/scheleaap/wmsnotes-desktop-and-server en https://github.com/scheleaap/wmsnotes-android voor meer informatie.
